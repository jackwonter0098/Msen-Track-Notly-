# Msen Track Notely

This is a Next.js application built with Firebase Studio that allows users to create, track, and manage personal challenges. Users can add daily notes, track their mood, and view their progress over time.

## Features

- **Challenge Management:** Create, edit, delete, archive, and duplicate challenges.
- **Progress Tracking:** View progress for each challenge.
- **Daily Notes:** Add daily notes with mood tracking for each day of a challenge.
- **Data Visualization:** View analytics and charts on your profile page.
- **Light/Dark Mode:** Theme toggling for user preference.
- **Responsive Design:** Fully responsive for both desktop and mobile devices.
- **Local Storage:** All data is saved in the browser's local storage, so no backend is required.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ShadCN UI
- Recharts for charts
- `react-hook-form` for forms
- `date-fns` for date manipulation

## Getting Started

To get started with this project:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
